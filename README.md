# visitorLogger
visitorLogger - A simple visitor logger for the website

Built with Flask and SQLite3.

## Installation

```bash
pip install flask
```

## Run the app

    $ python app.py


## Endpoints

    * `/` - Index, show endpoints in debug mode.
    * `/append` - Append a new visitor to the DB
    * `/post` - Append a new visitor to the DB using POST
    * `/clean` - Clean the DB
